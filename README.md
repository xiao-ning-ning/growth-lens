# 成长力场

> 知是行之始，行是知之成。先知己之所能，方知己之所向。

**让管理者的软性能力，从"感觉"变成"证据"。**

上传一段会议录音，AI 分析你的行为模式 → 量化能力维度 → 设计可落地的修炼路径。

## 解决什么问题

管理者面临的三大困境：

| 场景 | 痛点 |
|------|------|
| 述职季 | 下属写了 20 页 PPT，哪些是真正的能力，哪些只是话术？ |
| 晋升决策 | 凭印象打分，软性能力谁更强？说不清。 |
| 培训评估 | 花了冤枉钱送管理层上课，学了和没学有什么区别？ |

**成长力场的思路**：软性能力不是靠"问"，而是靠"观察行为"来推断。你说的话、做的事，比你对自己的评价更真实。

## 核心功能

### 1. AI 行为分析
拖入任意文字材料（会议记录、述职自评、访谈纪要），AI 自动提取：
- **已具备** — 你确实做到过的
- **待发展** — 你有这个意识但还不稳定
- **盲区** — "应该出现但你没意识到"的能力空白

每条结论都附**原文引用**，结论可追溯，不接受"感觉"。

### 2. 能力图谱
五大维度（战略与诊断、管控与绩效、人心与温度、知识与赋能、制度与设计）全局概览：
- 知道自己几斤几两
- 看到维度之间的关联关系

### 3. 核心组合
不是罗列能力点，而是识别"**1+1>2**"的化学反应。

比如：
- "手术刀式管理" = 战略拆解 × 理性决策
- "温度护城河" = 共情理解 × 长期信任

让优势**可命名、可传播、可复制**。

### 4. 修炼路径
针对每个待发展维度或盲区，AI 设计具体的修炼步骤——每一步都标注以**哪个已有能力作为杠杆**。

最小改变，最大效果。不需要推翻重来。

### 5. 成长轨迹
五维能力折线图，看得见的变化才有说服力。鼠标悬停查看任意日期所有分类的能力数量。

### 6. 团队管理视角（管理员）
一个页面查看团队所有人的成长轨迹，无需切换账号。

## 差异化

| | 成长力场 | 传统评估 | 在线 AI 平台 |
|---|---|---|---|
| 数据存储 | **本地**，不出公司 | 云端或纸质 | 云端 |
| 证据链 | **每条结论附原文** | 无 | 无 |
| 持续积累 | **增量生长**，不是每次重新分析 | 一次性的 | 一次性的 |

## 快速开始

### 1. 下载项目
```bash
git clone https://github.com/xiao-ning-ning/growth-force-field.git
```
或直接下载 ZIP 并解压。

### 2. 双击 start.bat
会自动检测并安装 Node.js（如未安装），安装依赖，启动服务。

### 3. 配置 API
首次启动会提示配置 API Key。创建或编辑项目根目录的 `.env` 文件：

```env
OPENAI_API_KEY=sk-your-api-key
```

支持的接口：OpenAI / DeepSeek / 月之暗面 / 本地 Ollama（任意 OpenAI 兼容接口均可）

浏览器打开 http://localhost:3000，默认账号 `admin / admin123456`。

## 技术栈

- **前端**：单 HTML 文件，深色主题，零构建
- **后端**：Node.js + Express
- **AI**：OpenAI 兼容接口（JSON mode）
- **数据**：本地 JSON 文件，按用户分目录存储
- **认证**：多用户 session + cookie
