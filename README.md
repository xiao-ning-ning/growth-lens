## 解决什么问题

管理者面临的三大困境：

场景 | 痛点 |
|------|------|
述职季 | 下属写了 20 页 PPT，哪些是真正的能力，哪些只是话术？ |
晋升决策 | 凭印象打分，软性能力谁更强？说不清。 |
培训评估 | 花了大把钱送管理层上课，学了和没学有什么区别？ |

**成长力场的思路**：软性能力不是靠"问"，而是靠"观察行为"来推断。你做的事、说的话，比你对自己的评价更真实。

## 核心功能

### 1. AI 行为分析
从会议记录、述职自评、访谈纪要中识别真实行为模式，量化已具备、待发展、盲区。每条结论都附**原文引用**，结论可追溯。

**星级评分**：每条证据标注正面（+1星）或负面（-1星），星级累加决定能力归属——正星→已具备，负星→待发展，星级归零→维度移除。双向流动，动态升降。

**录音预处理**：分析前自动做 ASR 纠错、语气词删除、断句修复，显著提升证据引用可读性。

### 2. 能力图谱
五维全局概览，看到能力全貌和各维度之间的关联。

### 3. 核心组合
把相关联的碎片能力打包成一个可解释的整体。知道它们是怎么配合的，强在哪里、什么时候用。

### 4. 盲区探测
AI 基于行为模式反推"你应该有，但自己没意识到"的能力——不是从录音中直接提取，而是通过跨维度对比、场景覆盖分析识别感知盲区。

### 5. 修炼路径
针对每个待发展维度或盲区，AI 设计具体的修炼步骤——每一步都标注以哪个已有能力作为杠杆。

最小改变，最大效果。不需要推翻重来。

### 6. 成长轨迹
五维能力折线图，追踪能力增长曲线。鼠标悬停查看任意日期的详细数据。

### 7. 自定义能力维度（管理员）
上传 Excel 定义自己的维度体系，AI 严格按你的标准分析，不受通用框架约束。

### 8. 团队管理视角（管理员）
热力图纵览团队能力分布，多用户对比折线图，无需切换账号即可制定个性化辅导策略。

## 快速开始

### 前置依赖

| 依赖 | 说明 |
|------|------|
| **Node.js** | 若未安装，Windows 按 Win+R → 输入 `msiexec /i https://nodejs.org/dist/v22.16.0/node-v22.16.0-x64.msi` 回车；macOS：`brew install node` |
| **.env 配置文件** | 首次启动时服务会自动创建默认配置，无需手动创建 |

> `start.bat` 会自动检测 `node_modules` 是否存在，若缺失则自动运行 `npm install`。`git clone` 下载包含 `node_modules`，ZIP 下载不包含但会自动补装。

### 启动步骤

```bash
# 1. 下载项目
git clone https://github.com/xiao-ning-ning/growth-force-field.git

# 2. 进入目录，双击 start.bat
# 自动检测端口占用、杀掉旧进程、启动服务、打开浏览器

# 3. 登录使用
# 打开 http://localhost:3000
# 默认账号：admin / admin123456
```

### 自定义配置（可选）

在项目根目录创建/编辑 `.env` 文件：

```env
OPENAI_API_KEY=sk-your-api-key
OPENAI_BASE_URL=https://api.deepseek.com
OPENAI_MODEL=deepseek-chat
ADMIN_PASSWORD=your-password   # 可选，不填则默认 admin/admin123456
```

**模型推荐**：优先使用 **DeepSeek**（性价比高、中文理解好、JSON 输出稳定）。支持 OpenAI / DeepSeek / 月之暗面 / 本地 Ollama（任意 OpenAI 兼容接口均可）。
