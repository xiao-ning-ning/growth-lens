> 知是行之始，行是知之成。先知己之所能，方知己之所向。

**让管理者的软性能力，从"感觉"变成"证据"**。

上传一段会议录音，AI 分析你的行为模式 → 量化能力维度 → 设计可落地的修炼路径。

## 解决什么问题

管理者面临的三大困境：

| 场景 | 痛点 |
|------|------|
| 述职季 | 下属写了 20 页 PPT，哪些是真正的能力，哪些只是话术？ |
| 晋升决策 | 凭印象打分，软性能力谁更强？说不清。 |
| 培训评估 | 花了大把钱送管理层上课，学了和没学有什么区别？ |

**成长力场的思路**：软性能力不是靠"问"，而是靠"观察行为"来推断。你做的事、说的话，比你对自己的评价更真实。

## 核心功能

### 1. AI 行为分析
从会议记录、述职自评、访谈纪要中识别真实行为模式，量化已具备、待发展、盲区。每条结论都附**原文引用**，结论可追溯。

### 2. 能力图谱
五维全局概览，看到能力全貌和各维度之间的关联。

### 3. 核心组合
把相关联的碎片能力打包成一个可解释的整体。知道它们是怎么配合的，强在哪里、什么时候用。

### 4. 修炼路径
针对每个待发展维度或盲区，AI 设计具体的修炼步骤——每一步都标注以哪个已有能力作为杠杆。

最小改变，最大效果。不需要推翻重来。

### 5. 成长轨迹
五维能力折线图，追踪能力增长曲线。鼠标悬停查看任意日期的详细数据。

### 6. 团队管理视角（管理员）
一个页面查看所有人的成长轨迹，无需切换账号。

## 快速开始

### 1. 下载项目
```bash
git clone https://github.com/xiao-ning-ning/growth-force-field.git
```
### 2. 双击 start.bat
自动检测并安装 Node.js（如未安装）、安装依赖、启动服务。

### 3. 配置 API
首次启动会自动生成安全密钥，无需手动配置。打开 http://localhost:3000，用默认账号 `admin / admin123456` 登录，登录后修改密码。

配置 API Key：编辑项目根目录 `.env` 文件，填入以下内容：
```env
OPENAI_API_KEY=sk-your-api-key
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4o
```
支持 OpenAI / DeepSeek / 月之暗面 / 本地 Ollama（任意 OpenAI 兼容接口均可）。
